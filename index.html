Hakkında

Freelancer, Start Bootstrap tarafından oluşturulan ücretsiz bir Bootstrap temasıdır. İndirilen dosyalar, HTML, CSS ve JavaScript dahil olmak üzere tam kaynak dosyalarını ve kolay özelleştirme için isteğe bağlı SASS stil dosyalarını içerir.

Kendi özel avatarınızı oluşturabilir, bölücülerdeki simgeleri değiştirebilir ve iletişim formunu tamamen işlevsel hale getirmek için e-posta adresinizi ekleyebilirsiniz!
